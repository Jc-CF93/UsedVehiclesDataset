# UsedVehiclesDataset
Aplicación web para visualizar la información de venta de automóviles usados

La primera parte muestra el Data Frame para que el usuario pueda revisar toda la información.
La siguiente sección se conforma por un casilla de verificación, que al ser seleccionada crea un histograma, el cual muestra la cantidad de vehiculos con relación al año del modelo.
La ultima sección se conforma por un boton que al ser presionado crea un gráfico de dispersión que muestra la relación del precio con el año del modelo.
